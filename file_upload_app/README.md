How to run this app

```
$ source django-venv/bin/activate // for mac
or
$ django-venv/bin/activate // for windows
```

```
$ pip install django
or
$ pip3 install django
```

```
$ pip install pandas
or
$ pip3 install pandas
```

```
$ cd file_upload_app
```

```
$ python file_upload_app/manage.py migrate
```

```
$ python file_upload_app/manage.py runserver
```
